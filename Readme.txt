To start the project :
                - Open your terminal and make sure you have the node and npm installed on your machine. 
                - In the terminal type the command and enter the command : 
                        npm start

 APIs:
 - Below are the apis for the queries/routes:
    
    UseCase:
        -Total word count
            API/ENDPOINT : /totalWords
            COMPLETE URL : http://localhost:4000/totalWords

        -Total number of sentences
            API/ENDPOINT : /totalSentences
            COMPLETE URL : http://localhost:4000/totalSentences

        -Number of occurrences of a particular word
            API/ENDPOINT : /totalOccurences
            COMPLETE URL : http://localhost:4000/totalOccurences?word=excepteur

        -10 most/least frequently occurring words (not fixed)
            API/ENDPOINT : /wordFrequency
            COMPLETE URL : http://localhost:4000/wordFrequency
